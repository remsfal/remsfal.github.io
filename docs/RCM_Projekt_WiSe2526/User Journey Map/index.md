title: 1.3 User Journey Map
---

In diesem Abschnitt werden die User Journeys der identifizierten Personas beschrieben.  Jede User Journey bildet den typischen Ablauf der Interaktion einer Persona mit dem System ab und dient als Grundlage für die Ableitung von Benutzeranforderungen und Use Cases.

Die detaillierten User Journeys sind jeweils auf eigenen Unterseiten dokumentiert und werden hier lediglich referenziert, um die Übersichtlichkeit der Gesamtdokumentation zu wahren.

## Übersicht der User Journeys

|  |  |  |
| --- | --- | --- |
| **Persona** | **Kurzbeschreibung** | **User Journey** |
| Handwerker/-in | Führt Reparatur- und Wartungsarbeiten durch und nutzt das System zur Organisation und Bearbeitung von Aufträgen | [Zur User Journey](https://xwiki.htw-berlin.de/bin/view/fb4wircm/2026%20WiSe%20Projekte/Team%20remsfal/User%20Journey%20Map/Handwerker-in/) |
| Mieter/-in | Meldet Mängel, verfolgt deren Bearbeitungsstatus und kommuniziert mit Vermieter:innen | [Zur User Journey](https://xwiki.htw-berlin.de/bin/view/fb4wircm/2026%20WiSe%20Projekte/Team%20remsfal/User%20Journey%20Map/Mieter-in/) |
| Vermieter/-in | Verwalten Immobilien, koordinieren Handwerker:innen und überwachen den Bearbeitungsfortschritt | [Zur User Journey](https://xwiki.htw-berlin.de/bin/view/fb4wircm/2026%20WiSe%20Projekte/Team%20remsfal/User%20Journey%20Map/Vermieter-in/) |
