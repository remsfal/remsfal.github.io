---

|  |  |  |
| --- | --- | --- |
| Phasen | 1. Schaden entdecken | 2. Information und Suche |
| Schritte / Aktionen | Linda bemerkt abends nach der Arbeit, dass die Heizung nicht funktioniert. | Sie überlegt, wie sie den Schaden melden kann.  Sie sucht die Telefonnummer oder E-Mail-Adresse der Hausverwaltung heraus. |
| Ziele / Erwartungen | Dass der Schaden so schnell wie möglich behoben wird | Die richtige Kontaktperson finden, um den Schaden melden zu können. |
| Herausforderungen / Pain Points | Ärger und Stress – besonders abends, wenn niemand erreichbar ist. | Telefonisch erreiche ich keinen und auf meine E-Mails kriege ich selten eine schnelle Antwort. |
| Informationen / Daten | Heizungsdefekt (Ort, Zeitpunkt, Auswirkung). | Telefonnummer oder E-Mail der Verwaltung. |
| Systeminteraktionen / Touchpoints | Wohnung, Smartphone. | Telefonverzeichnis, Mietvertrag, E-Mail-Postfach. |
| Verbesserungspotenzial / Anforderungen | Bessere Möglichkeiten zur Kontaktaufnahme auch nach Bürozeiten. | Klare Informationen im Mietvertrag oder Aushang im Haus, an wen man sich bei Schäden wenden soll. |
