| Die Analyse der Nachhaltigkeitsdimensionen basiert auf dem Drei-Säulen-Modell der Nachhaltigkeit, das ökologische, soziale und ökonomische Aspekte berücksichtigt. Die Aufgabe verfolgt das Ziel, die Auswirkungen auf den identifizierten User Stories und funktionalen Anforderungen auf Nachhaltigkeit zu bewerten und worauf bei der Umsetzung zu achten ist, um diese Aspekte zu unterstützen. Die Bewertung erfolgt anforderungsbezogen und orientiert sich an: der Reduktion von Ressourcenverbrauch (z. B. Papier, Wege, Zeit), der Entlastung und fairen Einbindung der beteiligten Stakeholder, sowie der Effizienz und Wirtschaftlichkeit der Prozesse. Es gilt: Eine Anforderung kann mehrere Nachhaltigkeitsdimensionen gleichzeitig berühren. | | | | |
